## Description

Brain Games is a set of five console games that train logical thinking and math skills. Each game is a small quest where you need to answer 3 questions correctly in a row.

## Games

- **Even** — Check if a number is even
- **Calculator** — Calculate the result of an expression
- **GCD** — Find the greatest common divisor
- **Progression** — Find the missing number in the progression
- **Prime** — Check if a number is prime

## Requirements

- Node.js 14+
- npm 6+

## Installation

```bash
git clone https://github.com/Linkoff/qa-auto-engineer-javascript-project-44.git
cd qa-auto-engineer-javascript-project-44
npm ci
npm link
```

## Usage

```bash
brain-even        # Check if a number is even
brain-calc        # Calculate the result of an expression
brain-gcd         # Find the greatest common divisor
brain-progression # Find the missing number in a sequence
brain-prime       # Check if a number is prime
```
